📋 System Instructions & Data Ingestion

  1. Download Schedule Data: Log in and download a schedule CSV file from Argos Student → Schedule Offering_Academic Courses Run Dashboard.
  2. Configure Report parameters: Select all the options desired and click on the Class Count Report button.
  3. Save Fields: Ensure you choose to save all fields to keep data structures intact for Banner compatibility.
  4. Sync with Application: In the scheduler interface, click Open via Direct Disk Sync and select the CSV file you have saved.

✨ Core Platform Features

  • Drag and Drop functionality: Effortlessly arrange and schedule courses visually on the interactive grid.
  • Granular Course Controls: Edit complete course details or seamlessly use the ADD tag to introduce new courses from scratch.
  • Conflict Prevention Engine: Instantly detects and warns about potential classroom/room collisions and faculty over-allocations.
  • Capacity Fill Gradients: Color-coded visual tiles represent enrollment density thresholds at 75%, 90%, and 100% capacity.
  • Multi-Dimensional Groupings: View and filter the full weekly calendar matrix by Week, Classroom, Professor, Campus, and Modality views.
  • Direct Auto-Save Pipeline: Updates your schedule catalog file in real time with an automatic changes indicator track column.
  • Banner-Ready Outputs: Create completely new course matrices from an empty canvas and download perfectly structured CSV catalogs for Banner entry.
